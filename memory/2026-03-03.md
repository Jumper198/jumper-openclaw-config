## 避坑与教训 (Lessons Learned)
- **子 Agent 监管**：发现子 Agent 在环境依赖报错时可能会产生进度误报。母体（赵无敌）已建立直接干预机制。
- **Discord UI 局限**：某些权限下内联按钮可能不可见，需通过底层配置突破。
- **升级路径**：对于 npm 全局安装的版本，需使用 `npm install -g openclaw@latest` 强行覆盖。
- **服务自愈**：发现升级后服务入口点（Entrypoint）未自动对齐（NVM vs Homebrew）。通过 `openclaw doctor --fix` 解决。
